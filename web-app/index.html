libp2p-kv-raft web client

This tab joins the raft cluster as a real, non-voting learner (over rust-libp2p, compiled to wasm) -- unlike the desktop CLI and the Android app it never votes in an election, but it does receive genuine AppendEntries replication and answers Get from its own locally-applied state, not just a request forwarded to a real voter.
